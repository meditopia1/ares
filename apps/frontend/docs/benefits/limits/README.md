# Benefit Limits

This folder contains benefit-cap and sub-limit reference material.

## Typical Topics

- annual benefit limits by product or plan
- per-event and per-visit caps
- family versus individual ceilings
- sub-limits such as hospital, dental, optical, or specialist allocations
- reset logic for annual or lifetime tracking

## Usage

Use this folder to store the business reference behind benefit-cap logic. Where limits are implemented in tables or validation code, the system source should remain authoritative and this folder should explain the business meaning.
