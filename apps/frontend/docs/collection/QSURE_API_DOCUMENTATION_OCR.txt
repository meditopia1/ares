INTRODUCTION
The QSCollectAPI is one of the file submission channels that QSURE(PTY) LTD offers to external
service providers who wish to import Debit Order and/or Disbursement files that are required
to be processed through the Xcelerate system.
This Channel allows service providers to interface their in-house systems directly and securely
with QSURE (PTY) LTD.
The API allows the following actions which uses the Standard Xcelerate File layout as well as the
One File Solution File Layout:
•
SUBMIT COLLECTION (DEBIT ORDER) FILES:
system.
This process submits a collection file to the API to be imported into the Xcelerate
SUBMIT PAYMENT (DISBURSEMENT) FILES
This process submits a payment file to the API to be imported into the Xcelerate system.
ndre

using the API. Any integration with the QSCollectAPI needs to ensure that when a user updates their password
on Xcelerate, the software integrating with the API uses the new password to authenticate the user.
